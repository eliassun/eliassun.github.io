#1. stash:
1.1 a specific file
git stash push -m "stash somethings changes" -- your file

1.2 stash all
git stash

1.3 pop the specific stash
  git stash list 
  git stash pop stash@{n}  , here n = 0, 1, ... 

1.4 pop the latest
  git stash pop
